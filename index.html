Chapter 1 Preface

Stories on a Sphere is a project that aims to communicate the threats imposed on the Arcticâ€™s marine ecosystem as a result of climate change, through the use of spherical presentation devices (Magic Planet and Science on a Sphere). This project was completed as a subset of UMBC Interdisciplinary CoLab, a summer program that provides undergraduate students the opportunity to conduct narrative-based research to create a final product. CoLab is only in session for four weeks, therefore imposing a major time constraint on this project. This is a description of how this project was completed within the four-week time period and serves as a guide for future researchers to create a similar product to be shown on Magic Planet.
